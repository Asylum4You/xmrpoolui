# XMRPool.net frontend

### AngularJS based UI for Furrypool?

### Features
- See your hashrate on all pages.
- Track multuple payment addresses.
- Hashrate siren when hashrate falls below a certain limit.
- Responsive

### Run it

Requires NodeJS

```sh
$ npm install
$ npm start
```
